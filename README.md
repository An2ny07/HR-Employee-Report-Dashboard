# HR Analytics Dashboard (MySQL + Power BI)
This project presents an interactive HR Employee Distribution Dashboard built using MySQL for data processing and Power BI for data visualization. It provides key insights into workforce demographics, employment trends, and organizational structure to support data-driven HR decisions.

## Project Overview
The dashboard analyzes employee data from 2000 to 2020, transforming raw HR records into meaningful visual insights. It focuses on workforce composition, retention patterns, and geographic distribution.

## Key Features
- Gender Distribution Analysis -
    Visual breakdown of male, female, and non-conforming employees. <br>
- Work Setup Insights (Headquarters vs Remote) - 
    Comparison of employees working on-site vs remote. <br>
- Average Length of Employment -
    KPI highlighting average tenure across the organization. <br>
- Employee Growth Trend (2000–2020) - 
    Time-series analysis showing workforce growth over the years. <br>
- Geographic Distribution - 
    Map visualization of employees across different states. <br>
- Department-Level Termination Rates - 
    Identification of departments with higher employee turnover. <br>
- Race Distribution Analysis - 
    Diversity insights across different racial groups.
